# Agente de IA - Extração de PDF com Power Automate Desktop

Estrutura de pastas:
- entrada_pdfs: Coloque o PDF a ser processado com o nome "entrada.pdf"
- saida_excel: A planilha extraída será salva aqui como "saida.xlsx"

Uso:
1. Copie o PDF alvo para "entrada_pdfs/entrada.pdf"
2. Execute o script Python `extrair_pdf.py`
3. O resultado será gerado em "saida_excel/saida.xlsx"

Pré-requisitos:
- Python instalado
- Java instalado
- Bibliotecas: tabula-py, pandas, openpyxl
